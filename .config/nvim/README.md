# how to use this nvim setup

make sure to have neovim version 0.12+

1. copy this nvim config folder to your .config dir
2. run nvim
3. install LSP of html, emmet_ls, ts_ls from :Mason 
4. now you have fully working autocomplete html:5 using emmet by blink.cmp
5. to activate another LSP snippet, just enable it by edit the [init.lua](./init.lua) file 

> the LSP directory comes from [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig/tree/master/lsp)
